Macro Intelligence API

This GitHub Pages site serves as the API proxy for the Macro Intelligence Terminal.

Available Endpoints

GET /api/fred.json

Get all available FRED data.

Example: ./api/fred.json

Data Sources

  • FRED - Federal Reserve Economic Data
  • Finnworlds - Financial and economic data

How to Update Data

To add or update data:

  1. Create a JSON file in the appropriate _data/api/[source] directory
  2. Commit and push the changes to the repository
  3. GitHub Pages will automatically rebuild the site

Sample Usage in JavaScript

fetch('https://elmooro.github.io/trading-dashboard/api/fred.json')
  .then(response => response.json())
  .then(data => {
    console.log('FRED Data:', data);
  })
  .catch(error => console.error('Error fetching data:', error));
