Macro Intelligence API

This GitHub Pages site serves as the API proxy for the Macro Intelligence Terminal.

Available Data

Data is stored in the /data directory of the repository and is accessible via the following URL pattern:

https://raw.githubusercontent.com/ElMooro/trading-dashboard/main/data/{api}/{endpoint}.json

Available APIs

  • fred - Federal Reserve Economic Data
  • finnworlds - Financial and economic data

Triggering Data Updates

To request fresh data, you can trigger the API proxy workflow manually:

gh workflow run api-proxy.yml -f api=fred -f endpoint=series/observations -f params='{"series_id":"GDP","observation_start":"2019-01-01"}'

API Documentation

FRED API

  • series/observations - Get observations for a specific series
  • series - Get information about a specific series

Finnworlds API

  • market-data - Get market data
  • economic-indicators - Get economic indicators
